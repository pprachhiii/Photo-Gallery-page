## 📸 Photo Gallery Page
A responsive photo gallery webpage featuring a sidebar menu with smooth transitions. The project uses HTML, CSS, and Font Awesome icons for an aesthetic and functional design.


### 🚀 **Live Preview**
[View Project](https://pprachhiii.github.io/Photo-Gallery-page/) 


### 🌟 **Features**
- Responsive sidebar navigation  
- Smooth sidebar toggle animation  
- Font Awesome icons integration  
- Attractive background image  
- Hover effects on menu items  
- Social media links  


### 🛠️ **Tech Stack**
- HTML5  
- CSS3  
- [Font Awesome](https://fontawesome.com/)  
- Google Fonts (Poppins)
  

### 💾 **How to use**
1. Clone the repository:  git clone git@github.com:pprachhiii/Photo-Gallery-page.git
2. Navigate to the folder:  cd Photo-Gallery-page   
3. Open `index.html` in your browser.
